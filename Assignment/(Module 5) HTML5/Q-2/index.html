Q-2. How to embed audio and video in a webpage?

ANS:-   To embed audio and video in a webpage, you can use HTML5's "audio" and "video" elements.

Add the "audio" element for audio: To embed an audio file, use the "audio" element. Inside the element, specify the source of the audio file using the src attribute.

The controls attribute adds basic audio controls (play, pause, volume) to the player, and the "source" element specifies the audio file's source.

Add the "video" element for video: To embed a video file, use the "video" element. Similar to the "audio" element, specify the video file's source using the src attribute.

The controls attribute adds basic video controls (autoplay, loop, muted) to the player, and the "source" element specifies the audio file's source.
