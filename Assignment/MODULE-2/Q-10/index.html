Q-10. Why should background and color be used as separate properties?

ANS:-   The background property is used to define the background color or image of an element. This property allows you to add visual interest to the background of an element or the entire page. By using the background property, you can set a solid color or image as the background of an element.

The color property, on the other hand, is used to define the text color or foreground color of an element. This property is useful for defining the color of the text, borders, and other foreground elements on a webpage. By using the color property.
